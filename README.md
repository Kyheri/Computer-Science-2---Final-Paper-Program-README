# Health Monitoring

# About the Project
This application allows users to monitor their health status through their device, providing ease of access which makes health monitoring more accessible. Tests like Glucose Tests, CBC, Urinalysis, Lipid Panel, Coagulation Tests, Thyroid Tests, and Kidney Function Tests can be used by the user to receive an evaluation of their overall health status. Feedback will be given to the user based on the data gathered by the application and as well as the possible risks and measures to take to improve their wellbeing.

# Objectives
     ● analyze data

     ● generate alerts
     ● provide health recommendations

# How to Run the Program

The app will start with showing TOS to inform the user, next would be asking if user has acknowledged and read the TOS, if the user answers no then they won’t be able to use the application, if the user answers yes they’ll be asked if they’d like to open the app or not.

if user wants to open the app
a set of prompts will appear asking about basic information like sex, and age.

After encoding the information a menu will appear asking which tests they would like to do.

===MENU===
1. Glucose Tests
2. CBC
3. Urinalysis
4. Lipid Panel
5. Coagulation tests
6. Thyroid Tests
7. Kidney Functions
8. EXIT

if the user chose any of the following a prompt will appear asking the user about certain information.
ex:
==Glucose Tests==
Enter your current blood glucose level in mg/dL: 

After the user encodes their input, advice will appear depending on the normal levels of that specific test.

ex:
==Glucose Tests==
Enter your current blood glucose level in mg/dL: 201
Advice: 
You're diabetic!! consult with a dietician to follow a balanced meal plan, 
limit your sugar intake and processed foods, increase your fiber intake, and
Exercise daily

After that the user will be taken back to the menu to choose between the options.
